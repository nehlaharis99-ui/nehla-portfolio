# Nehla Haris Portfolio

A modern, minimalist personal portfolio built with plain HTML, CSS, and JavaScript. It is ready for deployment on the Vercel free plan.

## Files

- `index.html` - Website content and sections
- `styles.css` - Design, layout, and responsive styles
- `script.js` - Mobile navigation and dynamic footer year

## Edit Your Details

Update these areas before final deployment if needed:

- Replace `resume.pdf` with your actual resume file in the project root.
- Add your GitHub or LinkedIn links in the contact section when available.
- Expand the SafeNest project with a live link, GitHub link, or screenshots.
- Change the Shapeza start year if `2022` is not correct.

## Run Locally

Because this is a static website, you can open `index.html` directly in a browser.

For a local server, use one of these commands:

```bash
python -m http.server 3000
```

Then open:

```text
http://localhost:3000
```

## Deploy to Vercel Free Plan

1. Create a GitHub repository, for example `nehla-portfolio`.
2. Upload these files to the repository:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
   - `resume.pdf` if you want the resume download button to work
3. Go to [Vercel](https://vercel.com/).
4. Sign in with GitHub.
5. Click **Add New Project**.
6. Import your portfolio repository.
7. Keep the framework preset as **Other**.
8. Leave build command empty.
9. Leave output directory empty.
10. Click **Deploy**.

Vercel will give you a free live URL like:

```text
https://your-project-name.vercel.app
```
